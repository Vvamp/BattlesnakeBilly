Meet Billy

Made by Vincent van Setten, student number 1734729

Billy is a battlesnake agent equipped with the powerful MCTS algorithm. It is slightly modified due to how battlesnake.com works, namely battling against multiple opponents at once and everyone moving simultaneously each turn.

Billy's brain computes how everyone moves and looks at the worst case scenario. He will try to win by attempting multiple random games for each possible move.

Get started with billy

  1. Simply run this webapp (which you've done if you see this text)
  2. Login to this webapp (custom accounts can be made in the LoginManager class, or use the account: 'Vvamp', with password 'admin').
  3. Optional: If you're on a local machine, make sure to port forward port 8080 and use your external ip as the website url mentioned below.
  4. Go to play.battlesnake.com
  5. Create a new battlesnake. Give it a name and set the server url to {whatever is in your webbrowser}/api/snakes/{your username}. For instance, when running on azure it could be 'https://eindopdracht-1718758585350.azurewebsites.net/api/snakes/Vvamp'
  6. Enjoy!
